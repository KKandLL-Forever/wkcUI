### API

#### tabs

属性 |	说明	| 类型 |	默认值
:--- | :--- | :--- | :---
`selected` | 当前选中的标签名 | String,Number | -

#### tabs-item

属性 |	说明	| 类型 |	默认值
:--- | :--- | :--- | :---
`name` | 标签 的名称，唯一且必填 | String,Number | -
`disabled` | title 禁用状态 | Boolean | false

#### tabs-content

属性 |	说明	| 类型 |	默认值
:--- | :--- | :--- | :---
`name` | 标签 的名称，唯一且必填 | String,Number | -
