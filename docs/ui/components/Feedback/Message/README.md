## API

属性 |	说明	| 类型 |	默认值
:--- | :--- | :--- | :---
`autoClose` | 是否自动关闭 | Boolean | true
`autoCloseDelay` | 自动关闭延时,单位秒 | Number | 5
`closeButton` | 关闭按钮 | text,callback | {text:'关闭',callback:undefined}
`position` | 提示框位置 | String | top
