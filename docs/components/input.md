## API

| 属性             | 说明                                           | 类型    | 默认值  |
| :--------------- | :--------------------------------------------- | :------ | :------ |
| `type`           | 声明 input 类型，同原生 input 标签的 type 属性 | String  | `text`  |
| `disabled`       | 是否禁用状态                                   | Boolean | `false` |
| `readonly`       | 是否只读状态                                   | Boolean | `false` |
| `value(v-model)` | 输入框内容                                     | String  | -       |
